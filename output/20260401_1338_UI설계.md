**WAYLO 국내 여행 AI 일정 추천 앱 화면 설계서**

### 1. 시작 화면

* **목적지 입력창**: 사용자가 여행 목적지를 입력할 수 있는 텍스트 입력창
* **검색 버튼**: 사용자가 입력한 목적지로 검색을 진행하는 버튼
* **최근 목적지 목록**: 사용자가 최근에 검색한 목적지 목록 (최대 5개)

예시 와이어프레임:
```
+---------------------------------------+
|  목적지 입력창  | 검색 버튼  |
+---------------------------------------+
|  최근 목적지 목록  |
|  1. 서울       |
|  2. 부산       |
|  3. 제주       |
+---------------------------------------+
```

### 2. 일정 생성 화면

* **AI가 생성한 3박 4일 일정 표시**: AI가 생성한 여행 일정 (날짜, 숙소, 맛집, 관광지 등)
* **지도 마커 표시**: 일정에 포함된 숙소, 맛집, 관광지 등의 위치를 지도에 마커로 표시
* **일정 변경 버튼**: 사용자가 일정을 변경할 수 있는 버튼

예시 와이어프레임:
```
+---------------------------------------+
|  일정 생성 결과  |
|  1일차: 서울   |
|  - 숙소: 호텔  |
|  - 맛집: 맛집  |
|  - 관광지: 관광지  |
+---------------------------------------+
|  지도 마커 표시  |
|  (숙소, 맛집, 관광지 위치 표시)  |
+---------------------------------------+
|  일정 변경 버튼  |
+---------------------------------------+
```

### 3. 숙소/맛집/관광지 카드 화면

* **상세 정보 표시**: 숙소, 맛집, 관광지의 상세 정보 (주소, 전화번호, 리뷰 등)
* **지도 보기 버튼**: 지도에서 위치를 확인하는 버튼
* **저장 버튼**: 사용자가 정보를 저장하는 버튼

예시 와이어프레임:
```
+---------------------------------------+
|  숙소/맛집/관광지 정보  |
|  이름: 호텔     |
|  주소: 서울시  |
|  전화번호: 02-1234-5678 |
|  리뷰: 4.5/5    |
+---------------------------------------+
|  지도 보기 버튼  |
|  저장 버튼       |
+---------------------------------------+
```

### 4. 지도 화면

* **카카오맵/네이버 지도 연동**: 지도 서비스와 연동하여 위치를 표시
* **마커 표시**: 숙소, 맛집, 관광지 등의 위치를 마커로 표시
* **위치 공유 버튼**: 사용자가 위치를 공유하는 버튼

예시 와이어프레임:
```
+---------------------------------------+
|  지도 화면       |
|  (카카오맵/네이버 지도 표시)  |
|  마커 표시 (숙소, 맛집, 관광지)  |
+---------------------------------------+
|  위치 공유 버튼  |
+---------------------------------------+
```

### 5. 마이 페이지

* **저장한 일정 목록**: 사용자가 저장한 일정 목록
* **최근 검색 목록**: 사용자가 최근에 검색한 목적지 목록

예시 와이어프레임:
```
+---------------------------------------+
|  저장한 일정 목록  |
|  1. 서울 3박 4일  |
|  2. 부산 2박 3일  |
+---------------------------------------+
|  최근 검색 목록  |
|  1. 서울       |
|  2. 부산       |
+---------------------------------------+
```
