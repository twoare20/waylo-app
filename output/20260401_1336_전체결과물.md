**WAYLO 국내 여행 AI 일정 추천 앱 화면 설계서**

### 1. 시작 화면

* **화면 ID:** start_screen
* **목적지 입력 필드:**
	+ 입력 필드 placeholder: "어디로 여행가고 싶으세요?"
	+ 입력 필드 타입: text
* **검색 버튼:**
	+ 버튼 텍스트: "검색"
	+ 버튼 타입: submit
* **최근 검색 기록 표시:**
	+ 최근 검색 기록 리스트: 
		- 최근 검색 기록 1
		- 최근 검색 기록 2
		- 최근 검색 기록 3
	+ 최근 검색 기록 최대 표시 개수: 3

```
+---------------------------------------+
|              목적지 입력 필드          |
|  (placeholder: 어디로 여행가고 싶으세요?)  |
+---------------------------------------+
|               검색 버튼                |
|               (텍스트: 검색)             |
+---------------------------------------+
|     최근 검색 기록 표시 영역           |
|  (최근 검색 기록 1, 최근 검색 기록 2, 최근 검색 기록 3) |
+---------------------------------------+
```

### 2. 일정 생성 화면

* **화면 ID:** itinerary_screen
* **AI 생성 일정 표시 (3박 4일):**
	+ 일정 날짜: 3박 4일
	+ 일정 활동 리스트: 
		- 숙소: 호텔 이름 (주소)
		- 맛집: 맛집 이름 (주소)
		- 관광지: 관광지 이름 (주소)
* **지도 마커 표시:**
	+ 지도 타입: 한국 지도
	+ 마커 표시 위치: 숙소, 맛집, 관광지 위치
* **숙소/맛집/관광지 카드 UI:**
	+ 카드 타입: 숙소, 맛집, 관광지
	+ 카드 내용: 
		- 숙소: 호텔 이름, 주소, 가격
		- 맛집: 맛집 이름, 주소, 메뉴
		- 관광지: 관광지 이름, 주소, 설명

```
+---------------------------------------+
|         AI 생성 일정 표시 영역         |
|  (일정 날짜: 3박 4일)                  |
|  (일정 활동 리스트: 숙소, 맛집, 관광지)  |
+---------------------------------------+
|               지도 표시 영역           |
|  (한국 지도, 마커 표시 위치: 숙소, 맛집, 관광지) |
+---------------------------------------+
|     숙소/맛집/관광지 카드 표시 영역     |
|  (카드 타입: 숙소, 맛집, 관광지)       |
|  (카드 내용: 호텔 이름, 주소, 가격 등)  |
+---------------------------------------+
```

### 3. 상세 정보 화면

* **화면 ID:** detail_screen
* **숙소/맛집/관광지 상세 정보 표시:**
	+ 상세 정보 타입: 숙소, 맛집, 관광지
	+ 상세 정보 내용: 
		- 숙소: 호텔 이름, 주소, 가격, 리뷰
		- 맛집: 맛집 이름, 주소, 메뉴, 리뷰
		- 관광지: 관광지 이름, 주소, 설명, 리뷰
* **이미지 표시:**
	+ 이미지 타입: 숙소, 맛집, 관광지 이미지
* **주소 표시:**
	+ 주소 타입: 숙소, 맛집, 관광지 주소
* **전화번호 표시:**
	+ 전화번호 타입: 숙소, 맛집, 관광지 전화번호
* **리뷰 표시:**
	+ 리뷰 타입: 숙소, 맛집, 관광지 리뷰

```
+---------------------------------------+
|     숙소/맛집/관광지 상세 정보 표시 영역  |
|  (상세 정보 타입: 숙소, 맛집, 관광지)    |
|  (상세 정보 내용: 호텔 이름, 주소, 가격 등) |
+---------------------------------------+
|               이미지 표시 영역          |
|  (이미지 타입: 숙소, 맛집, 관광지 이미지) |
+---------------------------------------+
|               주소 표시 영역            |
|  (주소 타입: 숙소, 맛집, 관광지 주소)    |
+---------------------------------------+
|               전화번호 표시 영역         |
|  (전화번호 타입: 숙소, 맛집, 관광지 전화번호) |
+---------------------------------------+
|               리뷰 표시 영역            |
|  (리뷰 타입: 숙소, 맛집, 관광지 리뷰)    |
+---------------------------------------+
```
