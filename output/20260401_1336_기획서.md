**WAYLO 국내 여행 AI 일정 추천 앱 기획서**

**개요**

WAYLO는 국내 여행을 위한 AI 일정 추천 앱입니다. 사용자는 목적지만 입력하면 AI가 자동으로 3박 4일 일정을 생성해줍니다. 지도 마커 표시, 숙소/맛집/관광지 카드 UI 등 직관적인 인터페이스를 제공하며, 한국어 전용으로 서비스를 제공합니다.

**화면 목록**

1. **시작 화면**
	* 목적지 입력 필드
	* 검색 버튼
	* 최근 검색 기록 표시
2. **일정 생성 화면**
	* AI 생성 일정 표시 (3박 4일)
	* 지도 마커 표시
	* 숙소/맛집/관광지 카드 UI
3. **상세 정보 화면**
	* 숙소/맛집/관광지 상세 정보 표시
	* 이미지, 주소, 전화번호, 리뷰 등
4. **마이 페이지**
	* 사용자 저장한 일정 표시
	* 사용자 프로필 정보 표시

**핵심 기능**

1. **목적지 입력 후 AI 일정 자동 생성**
	* 사용자는 목적지만 입력하면 AI가 자동으로 3박 4일 일정을 생성해줍니다.
	* AI는 사용자의 선호도를 고려하여 일정을 생성합니다.
2. **지도 마커 표시**
	* 생성된 일정에 포함된 숙소, 맛집, 관광지 등을 지도에 마커로 표시합니다.
	* 마커를 클릭하면 상세 정보 화면으로 이동합니다.
3. **숙소/맛집/관광지 카드 UI**
	* 일정에 포함된 숙소, 맛집, 관광지 등을 카드 형태로 표시합니다.
	* 카드를 클릭하면 상세 정보 화면으로 이동합니다.
4. **한국어 전용**
	* 모든 화면과 기능이 한국어 전용으로 제공됩니다.

**데이터 구조**

1. **사용자 데이터**
	* 사용자 ID
	* 사용자 프로필 정보
	* 저장한 일정 정보
2. **일정 데이터**
	* 일정 ID
	* 목적지
	* 생성 일자
	* 숙소, 맛집, 관광지 정보 등
3. **숙소/맛집/관광지 데이터**
	* 장소 ID
	* 이름
	* 주소
	* 전화번호
	* 이미지
	* 리뷰 등

**데이터베이스 설계**

* 사용자 데이터: 사용자 ID를 기본 키로 하여 데이터를 저장합니다.
* 일정 데이터: 일정 ID를 기본 키로 하여 데이터를 저장합니다.
* 숙소/맛집/관광지 데이터: 장소 ID를 기본 키로 하여 데이터를 저장합니다.

**API 설계**

* 사용자 등록/로그인 API
* 일정 생성 API
* 일정 조회 API
* 숙소/맛집/관광지 조회 API

**기술 스택**

* 프론트엔드: React Native
* 백엔드: Node.js, Express
* 데이터베이스: MongoDB
* AI 엔진: TensorFlow

**개발 일정**

* 1주일: 기획 및 설계
* 2주일: 프론트엔드 개발
* 3주일: 백엔드 개발
* 4주일: 데이터베이스 개발
* 5주일: AI 엔진 개발
* 6주일: 테스트 및 배포

**결론**

WAYLO 국내 여행 AI 일정 추천 앱은 사용자에게 편리하고 개인화된 여행 일정을 제공합니다. 한국어 전용으로 제공되며, 직관적인 인터페이스를 제공합니다. 개발 일정은 6주일로 예정되어 있으며, 기술 스택은 React Native, Node.js, Express, MongoDB, TensorFlow 등을 사용합니다.
