**WAYLO 국내 여행 AI 일정 추천 앱 기획서**

**개요**

WAYLO는 국내 여행을 위한 AI 일정 추천 앱입니다. 사용자는 목적지만 입력하면 AI가 자동으로 3박 4일 일정을 생성해 줍니다. WAYLO는 한국어 전용 앱으로, 국내 여행자들에게 최적화된 서비스를 제공합니다.

**화면 목록**

1. **시작 화면**
	* 목적지 입력창
	* 검색 버튼
	* 최근 목적지 목록
2. **일정 생성 화면**
	* AI가 생성한 3박 4일 일정 표시
	* 지도 마커 표시 (숙소, 맛집, 관광지)
	* 일정 변경 버튼
3. **숙소/맛집/관광지 카드 화면**
	* 상세 정보 표시 (주소, 전화번호, 리뷰 등)
	* 지도 보기 버튼
	* 저장 버튼
4. **지도 화면**
	* 카카오맵/네이버 지도 연동
	* 마커 표시 (숙소, 맛집, 관광지)
	* 위치 공유 버튼
5. **마이 페이지**
	* 저장한 일정 목록
	* 최근 검색 목록

**핵심 기능**

1. **AI 일정 생성 기능**
	* 사용자의 목적지 입력 후 3박 4일 일정 자동 생성
	* 일정에는 숙소, 맛집, 관광지 등이 포함됨
2. **지도 마커 표시 기능**
	* 지도에 숙소, 맛집, 관광지 마커 표시
	* 마커 클릭 시 상세 정보 표시
3. **숙소/맛집/관광지 카드 UI 기능**
	* 상세 정보 표시 (주소, 전화번호, 리뷰 등)
	* 지도 보기 버튼
	* 저장 버튼
4. **데이터 저장 기능**
	* 사용자의 일정, 숙소, 맛집, 관광지 정보 저장

**데이터 구조**

1. **일정 데이터**
	* 일정 ID (PK)
	* 사용자 ID (FK)
	* 목적지
	* 시작일
	* 종료일
	* 숙소 정보 (FK)
	* 맛집 정보 (FK)
	* 관광지 정보 (FK)
2. **숙소 데이터**
	* 숙소 ID (PK)
	* 이름
	* 주소
	* 전화번호
	* 리뷰 등
3. **맛집 데이터**
	* 맛집 ID (PK)
	* 이름
	* 주소
	* 전화번호
	* 리뷰 등
4. **관광지 데이터**
	* 관광지 ID (PK)
	* 이름
	* 주소
	* 전화번호
	* 리뷰 등

**기술 요구 사항**

1. **프론트엔드**
	* React Native 또는 Flutter를 이용한 모바일 앱 개발
	* 한국어 전용 앱
2. **백엔드**
	* Node.js 또는 Python을 이용한 서버 개발
	* 데이터베이스는 MySQL 또는 MongoDB 사용
3. **API**
	* RESTful API를 이용한 프론트엔드와 백엔드 통신
